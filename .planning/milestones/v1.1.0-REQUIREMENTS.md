# Requirements: Ontology Hub

**Defined:** 2026-01-27
**Core Value:** Enable wiki admins to go from local schema edit to GitHub PR in under 5 minutes, with zero platform accounts and strong validation feedback.

## v1.1.0 Requirements

Requirements for Dashboard and Resource entity types milestone.

### Dashboard Entity

- [x] **DASH-01**: Dashboard JSON schema defined in labki-ontology
- [x] **DASH-02**: Dashboard database table with pages JSONB column
- [x] **DASH-03**: Dashboard ingest from repo webhooks
- [x] **DASH-04**: Dashboard list/detail API endpoints
- [x] **DASH-05**: Dashboard graph visualization with document-shaped node
- [x] **DASH-06**: Dashboard detail page with pages accordion view
- [x] **DASH-07**: Dashboard create/edit form with wikitext editor
- [x] **DASH-08**: Modules can reference dashboards array
- [x] **DASH-09**: Bundles can reference dashboards array

### Resource Entity

- [x] **RSRC-01**: Resource JSON schema with additionalProperties:true
- [x] **RSRC-02**: Resource database table with category_key reference
- [x] **RSRC-03**: Resource ingest with hierarchical paths (like templates)
- [x] **RSRC-04**: Resource list/detail API endpoints with path support
- [x] **RSRC-05**: Resources queryable by category
- [x] **RSRC-06**: Resource graph visualization with form-shaped node
- [x] **RSRC-07**: Resource detail page with dynamic category fields
- [x] **RSRC-08**: Resource create/edit form with category-driven fields

### Auto-Derivation Chain

- [x] **DERV-01**: Properties with allowed_values.from_category auto-include referenced category
- [x] **DERV-02**: Categories in module auto-include their resources
- [x] **DERV-03**: Derivation chain handles cycles with visited sets
- [x] **DERV-04**: Draft patches use "add" op for derived arrays

### Integration

- [x] **INTG-01**: Draft CRUD for dashboards and resources
- [x] **INTG-02**: Resource field validation against category properties
- [x] **INTG-03**: PR submission includes dashboard/resource files
- [x] **INTG-04**: End-to-end derivation chain verified

## Out of Scope

| Feature | Reason |
|---------|--------|
| MediaWiki live sync | SemanticSchemas extension update separate project |
| Wikitext preview rendering | Would require MediaWiki parser; show raw for now |
| Resource versioning | Use repo version history |
